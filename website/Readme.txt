To run the app you need to
1- install laravel freamwork, https://laravel.com/docs/5.6/homestead
2- install database mysql,
3- create database 
DB_DATABASE=central21ch
DB_USERNAME=root
DB_PASSWORD=Ali_1986
4- import the database by
mysql -u root -p central21ch < central21ch.mysql
5- run the server
php artisan serve
